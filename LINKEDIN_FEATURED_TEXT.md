## Long Description (for GitHub About or LinkedIn article)

This project simulates a complete Workforce Management analytics cycle for a BPO contact center — the kind of analysis a WFM Director reviews every week to make staffing, routing, and coaching decisions.

Starting from a synthetically generated dataset (7,200 records · 80 agents · 90 days · 3 campaigns), the pipeline runs end-to-end in three Python scripts:

1. **Data generation** — realistic BPO logs with correlated metrics (adherence, shrinkage, AHT, CSAT, SLA, QA)
2. **KPI analysis** — six business questions answered with Pandas aggregations and exported as analysis-ready CSVs
3. **Chart generation** — six publication-quality Matplotlib charts + four Power BI–ready export CSVs

Every metric, KPI threshold, and business question reflects real operational experience from two years as a Real-Time Analyst in a global contact center. This is not a textbook exercise — it is the actual reporting loop, recreated in Python.

**Tech stack:** Python · Pandas · NumPy · Matplotlib · Jupyter · Power BI · Git
